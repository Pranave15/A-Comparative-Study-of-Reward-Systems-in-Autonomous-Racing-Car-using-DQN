# Optimizing Performance in Autonomous Racing Cars Using Deep Q-Learning: A Comparative Study of Reward Systems

## Overview

This repository contains the code and resources for our project on optimizing performance in autonomous racing cars using Deep Q-Learning. The project compares different reward systems to evaluate their impact on the performance of the autonomous racing cars in a simulated environment.
